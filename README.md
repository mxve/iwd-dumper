# IWD dumper
or: a very limited unzip with a very limited use case

![](https://i.imgur.com/laoMvjF.png)

- ```-s, --source <SOURCE>```
  - Source directory
- ```-o, --output <OUTPUT>```
  - Output directory
- ```-e, --extension <EXTENSION>```
  - Only dump this file extension [default: all]
- ```-t, --threads <THREADS>```
  - Concurrent threads to use (0 = system threads) [default: 0]
- ```-h, --help```
  - Print help
